FAQ

How do I install the template?

Simply copy and paste all the files in the code folder to your server.

How do I change the contact form information?

There is only on PHP file in the pack. Locate it, open it, and replace the email address there with your own.

How do I change the sidebar logo?

To change the sidebar logo, open style.css and go to line 126. There you will find the sidebar-logo class. Replace the image there with your own image, and give the background-size property to HALF of your logo size. This will make your logo look crisp on high resolution displays. Adgust the margin top until your logo is centered.  

To change the logo for the other pages, in style.css in line 361, locate .header .header-logo{} and do the same as above. 

How do I change icons?

The icon pack we used is font-awesome, its a simple <i class="fa fa-ICON-NAME"></i> that does the trick and creates amazingly beautiful icons. For the font-awesome cheatsheat go here http://fortawesome.github.io/Font-Awesome/icons/ and you'll find everything you need. In case you want to update the script, there is nothing easier. Go to the link provided and download the latest version. Replace the contents of the fonts folder with the new fonts downloaded and replace the font-awesome.css with the font-awesome.css you've just downloaded. Make sure to have a backup before doing so! 

Why are there two documentations?

One documentation is vital for you to understand the structure of the template, basically, to get you started, and the second documentation will show you all the features provided by our HTML, CSS and jQuery features.

Material Boxes

Containers are a little different from what you'll find in the second documentation because they have a material design. To create any element with a material design box give your container class the following attribute

<div class"container material-box">

</div>

To create a decoration line under one of these containers give it the following attribute

<div class="content decoration"></div>

 

Attention! 

The second documentation states you should have elements within a content class. While this is true for creating un material box design boxes, for material box design, you must have a container as explained above outside the content class.

Read More & Share Links

To create read more and share links inside your containers, simply add the following classes at the end of container material-box 




	<div class="container material-box">



	Your Stuff Goes Here



	 



	            <div class="more">



	                <a class="more-link" href="#">READ MORE</a>



	                <a class="share-link" href="#">SHARE</a>



	                <a class="link" href="#">VISIT GALLERY</a>



	            </div>



	            <div class="share">



	                <a href="https://www.facebook.com/sharer/sharer.php?u=LINK_HERE">FACEBOOK</a>



	                <a href="https://twitter.com/home?status=TEXT_AND_LINK_HERE">TWITTER</a>



	                <a href="https://plus.google.com/share?url=LINK_HERE">GOOGLE PLUS</a>



	            </div>



	</div>



	The system works quite simple. Clicking the share link will open the shareres with urls, replace LINK_HERE, TEXT_AND_LINK_HERE and LINK_HERE with your values. 



	In the more box, in order to add a static link, simply give it the link more-link or link. More link will show up colored, whilst link will show up black.
